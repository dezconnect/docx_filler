ms_filler
=====

Microsoft Word/Excel templates library for Erlang.

## Exported function

For docx files: 

  docx_template(TemplateFile, DestinationFile, Params)


For xls files: 

  xlsx_template(TemplateFile, DestinationFile, Params)


For plain files:

  template(TemplateFile, DestinationFile, Params)
